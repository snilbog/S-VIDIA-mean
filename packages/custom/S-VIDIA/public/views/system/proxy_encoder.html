Proxy Encoder

S‑VIDIA Proxy Encoder is an application used for reducing bandwidth from cameras when streaming video over a network. This software is installed on a PC near the edge device and used to encode one or more data heavy H.264 or MJPEG video streams to Flexible Delta Compression (FDC). The FDC stream can be viewed and recorded remotely using S‑VIDIA Client and Server software. The Proxy Encoder can be installed on virtually any Intel based PC with network connectivity. Encoder performance is dependent on CPU speed and camera resolution.

Example:

Steps to viewing a live image:

1. Install S‑VIDIA Proxy Server on virtually any modern Intel Based PC.
2. Setup h.264 or MJPEG camera within Proxy Server.
3. Connect to proxy server IP address via S‑VIDIA Client or Mobile Client.

To record video:

1. Follow above steps 1 and 2.
2. Connect to proxy server IP address via S‑VIDIA Server software.
